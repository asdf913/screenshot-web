# screenshot-web
## Build Command
```mvn clean package```
## Run Command
```java -jar servlet-java8-0.0.1-SNAPSHOT.jar```
### Port
```java -jar servlet-java8-0.0.1-SNAPSHOT.jar port=8081```
## URL
Screenshot
```http://127.0.0:8080/?title=eclipse-workspace```
### Image Format
```http://127.0.0:8080/?title=eclipse-workspace&format=jpeg```
### Image Dimension
```http://127.0.0:8080/?title=eclipse-workspace&w=50&h=100```
